Quick start


Welcome to signal++ v2.

Signal++ is an easy to use , header-only and open source signal/slot library for c++.

How to install signal++?
It's easy. just include signalpp.h in you'r code.

#include <signalpp.h>

When you connect a signal(e.g function a) to a slot(e.g function b) and using emit at end of you'r function a . you tell to signal++ when i called a , then you call b for me.
you can have many slots for a signal . if you connect 20 slots to signal a. when you called a . all of 20 slots will called.


Let's start signal++ with samples.
Connect a signal function to slot function:

#include <iostream>
#include "signalpp.h"
using namespace std;
using namespace signalpp;
void a()
{
cout<<"signal a called!"<<endl;
}
void b()
{
cout<<"slot b called!"<<endl;
}
int main()
{
connect(SIGNALFUNC(a),SLOTFUNC(b));// connect signal a() to b()
return 0;
}


connect(...) : using to connect a signal to a slot (and parameter is function name)
SIGNALFUNC(...) : using to specify signal function
SLOTFUNC(...) : using to specify slot function

and while program is running signal a() is connected to slot b()

Connect a signal function to slot member function:

#include <iostream>
#include "signalpp.h"
using namespace std;
using namespace signalpp;
void a()
{
cout<<"signal a called!"<<endl;
}
struct B
{
slots:
void b()
{
cout<<"slot b called!"<<endl;
}
};
int main()
{
B o1;
connect(SIGNALFUNC(a),SLOT(o1,b));
return 0;
}


slots: : using to specify slots area of struct or class ( it's same public and isn't important and using to make the code more readable! ).

Make signals responsible:
it's time to using signals and using emit

#include <iostream>
#include "signalpp.h"
using namespace std;
using namespace signalpp;
struct A
{
signals:
void a()
{
cout<<"signal a called!"<<endl;
emit(SIGNAL(*this,a));
}
};
struct B
{
slots:
void b()
{
cout<<"slot b called!"<<endl;
}
};
int main()
{
A o1;
B o2;
connect(SIGNAL(o1,a),SLOT(o2,b));
//now call o1.a to see the result
o1.a();
return 0;
}

emit( signal ) : this function tells signal++ that parameter signal is called
in this sample we used SIGNAL ( do not forget first parameter ; set it to *this not this).
and when we call that signal , slots will called.

you can use signals/slots and emit for previous samples.

How to specify signals/slot with non-void return type?! When you try to use SIGNAL|SLOT|SIGNALFUNC|SLOTFUNC with functions that doesn't have void return type compiler gives you errors. to fix error you need to change SIGNAL|SLOT|SIGNALFUNC|SLOTFUNC to SIGNAL2|SLOT2|SIGNALFUNC2|SLOTFUNC2 to specify custom return types.
patterns:

SIGNAL2(object,returntype,functionname)
SLOT2(object,returntype,functionname)
SIGNALFUNC2(returntype,functionname)
SLOTFUNC2(returntype,functionname)

sample:

#include <iostream>
#include "signalpp.h"
using namespace std;
using namespace signalpp;
int a()
{
cout<<"signal a called!"<<endl;
emit(SIGNALFUNC2(int,a));
return 0;
}
int b()
{
cout<<"slot b called!"<<endl;
return 0;
}
struct A
{
signals:
int a()
{
cout<<"signal A::a called!"<<endl;
emit(SIGNAL2(*this,int,a));
return 0;
}
};
struct B
{
slots:
int b()
{
cout<<"slot B::b called!"<<endl;
return 0;
}
};
int main()
{
A o1;
B o2;
connect(SIGNAL2(o1,int,a),SLOT2(o2,int,b));
connect(SIGNALFUNC2(int,a),SLOTFUNC2(int,b));
o1.a();
a();
return 0;
}


How to specify signals/slot with non-void argument types?! When you try to use SIGNAL|SLOT|SIGNALFUNC|SLOTFUNC|SIGNAL2|SLOT2|SIGNALFUNC2|SLOTFUNC2 with functions that have arguments compiler gives you errors. to fix error you need to add function argument types after SIGNAL|SLOT|SIGNALFUNC|SLOTFUNC|SIGNAL2|SLOT2|SIGNALFUNC2|SLOTFUNC2 parameters.
patterns:

SIGNAL(object,functionname,argumenttypes...)
SLOT(object,functionname,argumenttypes...)
SIGNALFUNC(functionname,argumenttypes...)
SLOTFUNC(functionname,argumenttypes...)


sample:

#include <iostream>
#include "signalpp.h"
using namespace std;
using namespace signalpp;
void a(double in)
{
cout<<"signal a called!"<<endl;
emit(SIGNALFUNC(a,double));
}
void b(int in)
{
cout<<"slot b called!"<<endl;
}
struct A
{
signals:
void a(int in)
{
cout<<"signal A::a called!"<<endl;
emit(SIGNAL(*this,a,int));
}
};
struct B
{
slots:
void b(float in)
{
cout<<"slot B::b called!"<<endl;
}
};
int main()
{
A o1;
B o2;
connect(SIGNAL(o1,a,int),SLOT(o2,b,float));
connect(SIGNALFUNC(a,double),SLOTFUNC(b,int));
o1.a(10);
a(10.0);
return 0;
}
Warning
You can't use function with more than 10 arguments in signal++.
Note
above codes also works with SIGNAL2|SLOT2|SIGNALFUNC2|SLOTFUNC2.

As you can see in output of above code, slots will not called.

WHY?!


becuase when you used emit(signal ) it's will call all slots that have not any arguments. but in above sample slots have parameters. to send parameters to slots you need to add some values after signals parameter of emit.
pattern :

emit( signal, parameters... )

sample :

1 emit ( signal , 10 )


in above sample after call all slots of signal parameter with (int) argument will called .like void test(int); void T::test(int) ; int t(int) and ... sample code:

How to change multy threading setting?

You can disable using SIGNALPP_T in you'r code before #include <signalpp.h>

#define SIGNALPP_T 0
#include <iostream>
#include "signalpp.h"
#include <thread>
using namespace std;
using namespace signalpp;
struct A
{
signals:
void a()
{
emit(SIGNAL(*this,a));
emit(SIGNAL(*this,a),10);
}
};
struct B
{
slots:
void b()
{
cout<<"slot B:b called\n";
this_thread::sleep_for(chrono::seconds(1));
cout<<"slot B:b done\n";
}
};
void a()
{
emit(SIGNALFUNC(a));
emit(SIGNALFUNC(a),10);
}
void b()
{
cout<<"slot b called\n";
this_thread::sleep_for(chrono::seconds(1));
cout<<"slot b done\n";
}
int main()
{
A _a;
B _b[7];
connect(SIGNAL(_a,a),SLOTFUNC(b));
for(int i=0;i<7;i++)
connect(SIGNAL(_a,a),SLOT(_b[i],b));
_a.a();
return 0;
}

When using #define SIGNALPP_T 0 multi threading feature will disable . default value is 1.

also You also can change maximum threads for 1 emit.

Other functions


disconnect(signal, slot): will disconnect a connected signal to slot.
sample:

#include <iostream>
#include <string>
#include "signalpp.h"
using namespace std;
using namespace signalpp;
void a()
{
cout<<"signal a called!"<<endl;
emit(SIGNALFUNC(a));
}
void b()
{
cout<<"slot b called!"<<endl;
}
int main()
{
cout<<"Before connect"<<endl;
a();
connect(SIGNALFUNC(a),SLOTFUNC(b));
cout<<"\nAfter connect"<<endl;
a();
disconnect(SIGNALFUNC(a),SLOTFUNC(b)); // just like connect
cout<<"\nAfter disconnect"<<endl;
a();// slot b will not called any more
return 0;
}

disconnectall(signal) : will disconnect all slots of a signal.
sample:

#include <iostream>
#include <string>
#include "signalpp.h"
using namespace std;
using namespace signalpp;
void a()
{
cout<<"signal a called!"<<endl;
emit(SIGNALFUNC(a));
}
void b()
{
cout<<"slot b called!"<<endl;
}
void c()
{
cout<<"slot c called!"<<endl;
}
int main()
{
cout<<"Before connect"<<endl;
a();
connect(SIGNALFUNC(a),SLOTFUNC(b));
connect(SIGNALFUNC(a),SLOTFUNC(c));
cout<<"\nAfter connect"<<endl;
a();
disconnectall(SIGNALFUNC(a));
cout<<"\nAfter disconnect"<<endl;
a();// slot b and c will not called any more
return 0;
}
